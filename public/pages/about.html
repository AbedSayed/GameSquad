ABOUT GAMESQUAD

Project Information

This project was developed as part of the Internet Technologies module at HTW Saar.

GameSquad is a web platform designed to help gamers connect, form teams, and organize gaming sessions. The application features real-time chat, friend management, and lobby creation functionality.

The platform utilizes modern web technologies including:

  • HTML5, CSS3 and JavaScript for frontend development
  • Node.js and Express for the backend server
  • Socket.IO for real-time communication
  • MongoDB for database management
